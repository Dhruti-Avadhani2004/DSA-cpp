VECTOR
vector is a dynamic array
once vector is filled, and you try to add an extra element, the vector doubles its size
naya vector banega jo purane ke double size ka hoga, saare elements copy honge aur puraana vector dump ho jaayega
#include<vector>
vector<int> v;
vector<int> last(a); --> to copy a vector to last named vector (name of vector is last and a)
vector<int> a(5, 1); --> 5 size vector and each space is initialized with 1
v.capacity();
v.push_back(1); -> v.capacity(); = 1
v.push_back(2); -> v.capacity = 2
v.push_back(3); -> v.capacity = 4
v.capacity(); --> 4 // total capacity of vector
v.size(); --> 3//no of elements in vector
v.front
v.back
v.pop_back --> get last element
v.clear --> size becomes 0 but capacity remains the same


DOUBLE ENDED QUEUE
AKA DECK
#include<deque>
deque<int> d;
d.push_back(1);
d.push_front(2);
d.pop_back();
d.pop_front();
d.at(1); // first index element
d.empty(); // empty the deque
d.erase(d.begin(), d.begin() + 1) // delete a range of or a single element in the deque

PRIORITY QUEUE -> first element is always the maximum in value
-> all elements entered (max heap / min heap) two ways to make the queue and it will pop in that order
#include<queue>
priority_queue<int> maxi; //for max heap;
priority_queue<int, vector<int>, greater<int>> mini; // min heap;
maxi.push(1);
maxi.size();
maxi.top(); // to access top element of heap
maxi.pop(); // element ko bahar nikal diya
// similar for mini also -> prints as min heap

DYNAMIC MEMORY ALLOCATION IN 2D ARRAYS
creating a 2d array
int arr[n][m]; //where n and m are constants
int * arr = new int[n]; --> using heap memory to create dynamic array 

objective: creating a 2d array based on input given by user at runtime
int *arr = new int[n]; --> creates an array for 1D

int **arr = new int* [n]; --> creating a 2D array of (n*n) size
##using a loop create the rows for each of these columns
for(int i = 0; i < n; i++){
    arr[i] = new int[n];
}

// creation done

for(int i = 0; i < n; i++){
    for(int j = 0; j < n; j++){
        cin>>arr[i][j];
    }
}

// taking input

for(int i = 0; i < n; i++){
    for(int j = 0; j < n; j++){
        cout<<arr[i][j]<<" ";
    }
    cout<<"\n";
}
